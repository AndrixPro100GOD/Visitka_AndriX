# Памятка по проекту Visitka_AndriX (портфолио)

## База проекта

- Проект полностью статический: `index.html` + `index.css` + папка `images/`.
- Сборка не нужна. Сайт открывается как `file://.../index.html` и публикуется на GitHub Pages.
- Для GitHub Pages главная страница — `index.html` в корне репозитория.

## Контент и структура

- Тексты и секции редактируются в `index.html` (опыт, образование, рекомендации, проекты, контакты).
- Список навыков в бегущей строке — массив `SKILLS_MARQUEE` в конце `index.html` (встроенный скрипт).
- Стили — в `index.css`.
- Фото профиля: `images/photo.png` (при смене файла обновите query-параметр версии в `index.html`).
- Рекомендательное письмо (PDF): `documents/recommendation-koksharov.pdf` (латинское имя файла — стабильные URL на GitHub Pages).
- Внутренние переходы — якоря (`#about`, `#skills`, `#experience`, `#education`, `#references`, `#projects`, `#contact`).

## Кэш и обновления

- При каждом релизе менять версию в query-параметрах:
  - `index.html`: `index.css?v=...`
  - при смене фона/картинок в CSS — соответствующие `?v=...`
- Формат: `v=YYYYMMDD-N` (пример: `v=20260512-1`).
- После деплоя проверить в инкогнито и с принудительным обновлением (Ctrl+F5).
