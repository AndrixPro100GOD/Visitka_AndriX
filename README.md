# Портфолио — статический сайт

Проект без сборки: откройте `index.html` в браузере или опубликуйте на [GitHub Pages](https://pages.github.com/).

## Локально

- Откройте `index.html` двойным щелчком, либо поднимите любой статический сервер в корне репозитория.

## GitHub Pages

1. Загрузите репозиторий на GitHub.
2. **Settings → Pages**: источник — ветка `main`, папка **/ (root)**.
3. Главная страница должна называться `index.html`.

Подробные правила проекта — в [documents/PROJECT_RULES.md](./documents/PROJECT_RULES.md).
